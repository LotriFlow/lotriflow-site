# BreatheFree Timer

A progressive web app to help you quit smoking by tracking your smoke-free time, managing cravings, and celebrating milestones.

## Features

- **Timer Tracking**: Real-time countdown to your next cigarette break
- **Milestones**: Track health improvements as time passes
- **Achievements**: Unlock rewards for staying smoke-free
- **Statistics**: View money saved, cigarettes avoided, and streaks
- **Reports**: Weekly/monthly charts of your progress
- **AI Coach**: Personalized encouragement and tips
- **Breathing Exercises**: Guided breathing to manage cravings
- **PWA Support**: Install as an app on mobile devices
- **Offline Support**: Works without internet connection
- **Data Export/Import**: Backup and restore your progress
- **TEST**
## Getting Started

1. Open `index.html` in your web browser
2. Complete the initial setup (daily limit, target interval, pack price)
3. Start your smoke-free journey!
